Modèles retenus

Cette page présente les modèles de segmentation d'images que nous avons développés pour ce projet.

Pour chaque modèle, nous présentons rapidement son architecture, ainsi que les valeurs de métriques de performance (accuracy, IoU score et F1-score) obtenues sur le jeu de données de test.

Modèle U-Net avec VGG16

Ce modèle utilise le modèle de type encoder/decoder : U-Net. Le modèle utilise un squelette inspiré de VGG16 pour améliorer le modèle U-Net intial.

Modèle FPN avec VGG16

Ce modèle utilise la méthode Feature Pyramid Network (FPN). Le modèle utilise un squelette inspiré de VGG16 pour améliorer le modèle FPN.
